Embedded Project --> Pico Pirate

This code integrated Buddy 2, Buddy 4 and Buddy 5 for Week 10 presentation.

main.c is the main file where all the tasks are created, and this is running on FreeRTOS.

CGI: Common Gateway Interface to allow running scripts externally with handlers e.g. "/debug.cgi?debug=1"

SSI: Server-Side Interface allows for dynamic data to be entered into the HTML.

To visit webpage: http://<IP>

The values for frequency and duty cycles captured are for the last 10 pulses.
